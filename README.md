Alien Invasion Final Project

A modified copy of the alien invasion participation repo worked on in class, forked from [the starter repo](https://github.com/RedBeard41/alien_Invasion_starter.git) 
Pursuing the modified gameplay pathway using the assets from the starter repo.

Gameplay Modifications added:
1. Starting ship position randomized to any x-coordinate at the bottom of the screen
2. Ship able to be controlled with WASD and arrow Keys
3. Ship can tilt 30 degrees left or right (using q and e keys) and fire bullets diagonally, when not tilting ship slowly returns to facing straight up
4. Ship now capable of vertical movement (up to halfway up the game screen)

What else I'd like to (and plan to eventually) add for the ship:

1. Machine gun blaster (player can hold spacebar to fire a volley of bullets up to roughly 20 before a cooldown)
2. Charge shot (player holds spacebar for about one second to charge a blaster bolt which will be capable of hitting 2 alien ships before disappearing)
