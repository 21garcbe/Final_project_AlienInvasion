Alien Invasion Final Project

A modified copy of the alien invasion participation repo worked on in class, forked from [the starter repo](https://github.com/RedBeard41/alien_Invasion_starter.git) 
Pursuing the modified gameplay pathway using the assets from the starter repo.

Gameplay Modifications added:
1. Starting ship position randomized to any x-coordinate at the bottom of the screen
2. Ship able to be controlled with WASD and arrow Keys
3. Ship can tilt 30 degrees left or right (using q and e keys) and fire bullets diagonally, when not tilting ship slowly returns to facing straight up
4. Ship now capable of vertical movement (up to halfway up the game screen)
5. Alien Fleets change shapes randomly each successive level (betwen rectangle, triangle, "m" shape)
6. "Tough" aliens added with 20% spawn rate, neeed hit twice to die. Color coded purple for easy identification
7. A store menu system activated with the "m" key which displays possible upgrades and pauses gameplay while active and drawn to the screen. Menu updates with new purchases to show what
items are owned
8. Added spendable credits tied to score that increase with aliens killed and can be spent 
at store menu to get upgrades
9. Machine gun upgrade available for purchase in the store menu, uses cooldown system to 
make using it more dynamic with bursts requiring a 1 second cooldown 
10. Added rotating arsenal activated by pressing "tab" key to rotate between machine gun and 
regular blaster (after the upgrade is purchased)
